개요 jQuery는 Java Script 언어를 간편하게 사용할 수 있도록 단순화시킨 오픈 소스 기반의 js라이브러리이다. jQuery는 문서 객체 모델(DOM)과 이벤트(EVENT)에 관한 처리를 손쉽게 구현할 수 있다. 또한, Ajax 응용 프로그램 및 플러그인도 빠르게 개발할 수 있도록한다. 적용 jQuery는 js라이브러리이므로 (.js 파일) 형태로 존재한다. jQuery를 사용하기 위해서는 jQuery파일을 먼저 웹페이지에 로드해야한다. 파일 삽입 방법 CDN을 로드하는 방법 CDN이란? CDN이란 웹사이트의 접속자가 서버에서 콘텐츠를 다운받아야 할 때, 자동으로 가장 가까운 서버에서 다운받도록 하는 기술이다. 이는 서버에 트래픽이 집중되지 않고, 콘텐츠 전송시간이 빨라진다.
