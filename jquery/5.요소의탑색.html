트리 탐색(tree traversing) : 특정 요소로부터 다른 요소들과의 관계를 통해 선택하길 원하는 요소까지 dom트리를 검색해 나아가는 과정. - 조상(ancestor)요소 탐색 - 형제(sibling)요소 탐색 - 자손(descendant)요소 탐색 조상(ancestor)요소의 탐색 .parent() 메소드 : 선택한 요소의 부모요소를 선택한다.

          .parents() 메소드 : 선택한 요소의 조상요소를 모두 선택한다. 인자에 값을 넣으면( ex. .parents("div") ) 조상 요소중 인자에 해당하는 것들만 효과가 적용된다. parentsUntil() 메소드 : 선택한 요소의 조상 요소 중에서, 전달받은 선택자에 해당하는 요소 바로 이전까지의 요소만을 모두 선택한다. $(function () {             $("button").on("click", function () {                 $("p").parentsUntil(".container").css({                     "border": "1px solid red"                 })             })         }) .closest() 메소드 : 자신을 포함한 조상 요소 중에서 전달받은 선택자에 해당하는 요소의 집합에서 가장 첫 번째 요소를 선택한다. 형제(sibling) 요소의 탐색 .siblings() 메소드 : 선택한 요소의 형제요소 중에서 지정한 선택자에 해당하는 요소를 모두 선택한다.

          .next() 메소드 :선택한 요소의 바로 다음에 위치한 형제 요소를 선택 .nextAll() 메소드 : 선택한 요소의 다음에 위치한 형제요소를 모두 선택한다. .nextUntil() 메소드 : 선택한 요소의 형제요소 중에서 전달받으느 선택자에 해당하는 요소 바로 이전까지의 요소를 모두 선택한다. .prev() .pevAll() .prevUntil() : 상위의 것들과 반대로 작동한다. 자손(descendant)요소의 탐색 .children() 메소드 : 선택한 요소의 자식요소를 모두 선택한다. 이때, 선택자를 인수로 전달하여, 전달받은 선택자에 해당하는 자식요소만을 선택할수도 있다.          div요소

          .find() 메소드 : 선택한 요소의 자손중에서 전달받은 선택자에 해당하는 자손요소를 모두 선택한다. 기타 탐색 메소드 .add() 메소드 : 선택한 요소의 집합에 전달받은 요소를 추가한다. 선택요소를 확장하는 개념이다. $("h1").css("background-color","yellow"); $("h2").css("background-color","yellow"); //  또는 $("h1, h2").css("background-color","yellow"); 와 같은 것을, $("h1").add("h2").css("background-color","yellow"); 라 표현 할 수 있다. .each() : 선택한 요소 집합의 요소마다 전달받은 콜백함수를 실행한다.
