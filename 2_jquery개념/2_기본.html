문법 : jQuery를 사용하여 간편하게html 요소를 선택하고, 선택된 요소에 특정 동작을 설정할 수 있다. 달러($)기호는 jQuery를 의미하는 식별자이다. $(선택자).동작함수(); $() 함수 : 선택된 html 요소를 jQuery에서 이용할 수 있는 형태로 생성해 주는 역할을한다. $() 함수의 인수로는 html 태그 이름뿐 아니라 css선택자를 전달하여 html요소를 선택할 수 있다. 이러한 $() 함수를 통해 생성된 요소를 제이쿼리 객체(jQuery object)라고 한다. jQuery는 이렇게 생성된 jQuery object의 메소드를 사용하여 여러 동작을 설정할 수 있다. Document객체의 ready()메소드 : js 코드는 웹브라우저가 문서의 모든 요소를 로드한 뒤에 실행되어야한다. 하지만, - 아직 생성되지 않은 html 요소에 속성을 추가하려고 할 경우 - 아직 로드되지 않은 이미지의 크기를 얻으려고 할 경우 와 같은 경우에는 오류가 발생한다. 이러한 문제 때문에 js에서는 window객체의 onload() 메소드를 이용하여 문서가 모두 로드된 뒤에 코드가 실행되도록 설정한다. window.onload() {     자바스크립트 코드; } jQuery에서는 document객체의 ready() 메소드를 이용하여 같은 결과를 보장한다. $(document).ready(function () {     제이쿼리 코드; }); jQuery Team 에서의 방법 $(function () {     제이쿼리 코드; }); document객체의 메소드인 ready()함수를 사용할 때와 안할때의 차이 예제
